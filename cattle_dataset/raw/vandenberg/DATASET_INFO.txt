Van den Berg et al. Dataset Structure
=====================================

Dataset DOI: 10.5061/dryad.rq80k
Location: cattle_dataset/raw/vandenberg/

Files:
- Genotypes_26503SNPs.txt (33 MB): Genotype data with 26,503 SNPs
- ID_Breed.txt (11 KB): Breed identifiers for samples

Phenotype Data:
Organized by genetic correlation (GenCor). This repo ships the r_g = 0.8
scenario used by the poster; the r_g = 0.4 and 1.0 scenarios are available
from the same Dryad source (DOI: 10.5061/dryad.rq80k).
- Phenotypes_GenCor_0.8/ (100 replicate files)

The Phenotypes_GenCor_0.8/ directory contains 100 replicate files:
- Phenotypes_replicate_1.txt
- Phenotypes_replicate_2.txt
- ...
- Phenotypes_replicate_100.txt

Total files: 1 genotype + 1 ID + 100 phenotype replicates = 102 files
